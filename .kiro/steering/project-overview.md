# 분실물 찾아주기 프로젝트 (Lost & Found AI Matching Platform)

## 프로젝트 개요
비전 및 텍스트 기반 AI를 활용한 분실물 매칭 플랫폼.
제보자가 습득한 분실물을 등록하면, AI가 자동으로 캡션을 생성하고,
찾는 사람의 텍스트 설명과 유사도를 비교하여 매칭해주는 시스템.

## 핵심 기능

### 1. 분실물 제보 (발견자)
- 분실물 사진 업로드 + 카테고리 선택 (지갑/카드/신분증/전자기기/기타 등)
- AI가 이미지 캡션 자동 생성 (비전 모델)
- 발견 장소, 시간 등 부가 정보 입력

### 2. 분실물 검색 (찾는 사람)
- 텍스트로 분실물 설명 입력
- AI Agent가 텍스트-텍스트 유사도 기반으로 매칭 (캡션 vs 설명)
- 카테고리별 브라우징 지원

### 3. 채팅 및 교환
- 매칭된 발견자 ↔ 찾는자 간 채팅 주선
- 만남 예약 기능
- 교환 완료 후 "찾기완료" 상태 표시

### 4. 회원가입 및 본인인증
- 전화번호, 주민등록번호 기반 본인인증
- 학교 인증 (학교 이메일 등)
- 지갑/카드/신분증 분실물의 경우, 회원가입 시 작성한 이름 일치 여부 확인 후 진행

## 기술 스택 가이드
- 언어: Python
- AI/ML: 이미지 캡셔닝 모델, 텍스트 유사도 모델 (sentence-transformers 등)
- 백엔드: FastAPI 또는 Flask
- 데이터베이스: PostgreSQL 또는 SQLite (개발 단계)
- 채팅: WebSocket 기반 실시간 통신

## 차별점
- AI 기반 자동 매칭으로 분실물 찾기 간편화
- 카테고리별 분류로 접근성 향상
- 본인인증 시스템으로 신뢰성 및 안전성 확보
- 신분증류 분실물에 대한 이름 매칭 검증

## 코딩 컨벤션
- Python PEP 8 준수
- 타입 힌트 사용
- docstring 작성 (Google style)
- 한국어 주석 허용, 코드/변수명은 영어
- API 응답은 JSON 형식, 한국어 메시지 포함
